# 读书整理 Agent · 架构总纲

> 与「电影整理 agent」同构的六层解耦架构（见 电影 Agent/00-架构总纲.md），只把领域从电影换成书：
> 输入「一句话『我读了 xx』/ 书封截图 / 手填」，agent 自动补全成带多维标签的**藏书票**（作者 / 译者 / 类型 / 流派 / 剧情 / 我的评分），
> 由地理子 agent 判**故事地 / 作者地**，建议钉到地球。解耦在 `src/app/lib/book/`，不动 FROST 内核，只当 userMarks 总线生产者。

## 数据契约（BookTags ⟂ GeoTarget）

- **BookTags**：`author / translator / genre(类型) / movement(流派/文学运动) / plot(剧情梗概一句) / userRating(我的评分)`
- **GeoTarget**：`kind: 'story'(故事地) | 'author'(作者地) | 'country'(国家兜底)` + `place / lng / lat / confidence`

## 六层（对标电影 agent，文件同名）

| 层 | 文件 | 职责 |
|----|------|------|
| 感知 | `sense.ts` | 一句话抽书名+评分（《》优先 / 去「读了·在读·读完·标记」噪声）；书封截图端侧 OCR（原图不出端） |
| 记忆·知识 | `catalog.ts` | 本地豆瓣读书库当 RAG：归一书名模糊匹配 → 作者/国家/年份/评分/简介 |
| 推理 | `resolve.ts`(并入 agent) | 兜底链：本地库→云脑补全→手填 |
| 协作·子agent | `tagging.ts` | ①补全子agent（云脑 JSON：作者/译者/类型/流派/剧情 + 故事地/作者地）②地理子agent（故事地→作者地→国家逐级 geocode） |
| 反思 | `critic.ts` | 评分钳、年份合理、坐标合法、幻觉剔除、应用用户历史纠错 |
| 行动 | `pin.ts` | suggest-then-confirm：确认才 `addUserMark(kind:'book')`，全标签 meta + 喂画像 |
| 编排 | `agent.ts` | 串流水线 + onPhase；`confirmPin` |

## 兜底链

```
一句话「我读了《xx》几星」/ 书封截图 OCR / 手填 → 候选书名
  ① 本地豆瓣读书库模糊匹配（确定锚点：作者/国家/年份/评分/简介）
  ② 云脑补全子agent（译者/类型/流派/剧情 + 故事地/作者地）
  ③ 手填骨架（needsConfirm）
  ④ 地理子agent：故事地→作者地→国家（geocodeCity / bookCountry）
  ⑤ 都没有 → needPlace，存藏书票、等用户手选地点
```

## 协作层第二个子 agent：读书笔记结构化（notes.ts）

> 「不管用户输入什么，都整理成结构化读书笔记」。`src/app/lib/book/notes.ts`。
- 输入：任意零散文字 / 书页·手写截图（端侧 OCR，原图不出端）。
- 云脑 JSON 一次成型：`{ bookTitle, themes[], quotes[](原文摘抄), insights[](读者感想), characters[], links[](关联书/概念), summary }`——强约束「区分摘抄与感想、忠实原文不编造」。
- **舱壁兜底**：云脑不可用 → 本地按行粗分（带引号/破折号的归摘抄，其余归感想），至少不丢原文。
- 端侧持久化 `pe.bookNotes.v1`（localStorage 发布订阅）；`getNotesForBook(title)` 可按书聚合，未来挂到藏书票详情。
- UI：BooksRunPage 顶部「标记书 / 整理笔记」模式切换；笔记草稿卡分区显示 主题/摘抄/感想/人物/关联/小结，确认存进笔记本。
- 已验证：《活着》散记 → 自动识别书名 + 区分摘抄(原文金句)与感想 + 关联《许三观卖血记》；《看不见的城市》→ 主题×3/摘抄/人物/小结。

## 与设施接线

- 落点 `addUserMark({kind:'book', meta})` → 地球紫点；`MarkerDetail` book 分支扩展显示 类型/流派/译者 + 故事地落点。
- 运行页增强 `BooksRunPage` 的「记一本」→ agent 出草稿藏书票，确认才钉；保留手填兜底。
- 复用 `geoStickers.geocodeCity`（城市级，与电影共用）+ `books.bookCountry`（国家兜底）。
